# CashBook Application

A premium, dark-themed cashbook application built with Next.js and Tailwind CSS.

## Features

-   **Multiple Books**: Create separate books for different accounts (e.g. Wallet, Bank, Saver).
-   **Fixed Books**: Support for asset-based books where balance is updated directly (snapshot), perfect for cash counting.
-   **Global Date Filter**: View your net worth or activity for a specific day across all books.
-   **Transaction Tracking**: Add income and expense entries.
-   **Real-time Balance**: Automatically calculates total balance, total income, and total expenses.
-   **History Log**: View a list of past transactions with dates and descriptions.
-   **Data Persistence**: Uses LocalStorage to save your data automatically.
-   **Dark Glass Aesthetic**: Modern UI with glassmorphism effects and deep dark gradients.

## Tech Stack

-   **Framework**: Next.js 15 (App Router)
-   **Styling**: Tailwind CSS
-   **Language**: TypeScript

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
4.  Open [http://localhost:3000](http://localhost:3000) with your browser.
